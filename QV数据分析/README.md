|[毕业生毕业去向分析]()|1.分析各学院毕业生毕业之后的去向，给毕业生提供参考。  2.对比不同家庭状况（城镇或农村）、性别的去向比例，给不同家庭状况的人提供参考。        3.分析了高考成绩、招生计划人数和毕业方向的关系，可以给想要报考河科大的高考学生提供参考||已完成|
