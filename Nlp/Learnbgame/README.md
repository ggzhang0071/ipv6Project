# 功能介绍：
            学术论文大多都是PDF格式的，该项目旨在从PDF文件中提取相关内容，
            并将相关内容训练学习模型，力求实现一个学术型机器人的建立。
            面向github，实现一个程序学习为主的机器人

# 相关模块的安装
            pip install -r requirements.txt
